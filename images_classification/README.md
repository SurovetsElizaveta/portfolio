# Классификация изображений с помощью нейронной сети

### Описание проекта
Цель - классифицировать изображения, построив свёрточную нейронную сеть

### Выводы
- Обучены две нейронные сети - простая CNN сеть с 5 слоями и готовая модель ResNet18 из библиотеки torchvision
- Построены графики значений функции потерь и метрики точности по ходу обучения
- Так как модель ResNet18 показала лучшие значения точности, построим предсказание по тестовым данным с помощью этой модели
- Результат точности предсказаний по тестовой выборке: 0.93517
